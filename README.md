# iCarros-MVVM
Teste de MVVM para o iCarros

Esse projeto foi feito para o auxílio de entendimento da arquitetura MVVM, ele foi desenvolvido usando xCode 11 (BETA). 
Com isso é preciso ter uma versão igual o superior do xCode 11. 
Ele também já esta preparado para receber Swif UI, com isso não funciona em aparelhos com iOS menores que o 13.
